# Klasifikace kreditního skóre – bakalářská práce

Tento repozitář obsahuje finální Jupyter Notebook a doplňkové soubory k bakalářské práci zaměřené na analýzu úvěrového rizika a klasifikaci kreditního skóre.

## Doporučené prostředí

Doporučeným prostředím pro prohlížení i spuštění notebooku je platforma Kaggle, na které byl notebook primárně vyvíjen a testován.

## Reprodukovatelnost

Pro reprodukci workflow je nutné nahrát požadované datové soubory a nainstalovat závislosti uvedené v souboru `requirements.txt`.

Notebook je doporučeno spouštět postupně od začátku do konce.

## Alternativní zobrazení notebooku

Notebook je primárně určen pro prostředí Kaggle, ale pro statické prohlížení je dostupný také přes GitHub a nbviewer:

- GitHub: [Notebook na GitHubu](https://github.com/karolina666-it/bachelor-thesis-credit-scoring/blob/main/credit-score-classification-model.ipynb)
- nbviewer: [Notebook v nbvieweru](https://nbviewer.org/github/karolina666-it/bachelor-thesis-credit-scoring/blob/main/credit-score-classification-model.ipynb)

## Omezení alternativních prostředí

GitHub a nbviewer slouží především ke statickému zobrazení notebooku. Některé pokročilejší vizuální nebo interaktivní prvky se proto nemusí vykreslit stejně jako v Kaggle.

Zejména sekce **13) Predikce nového žadatele pomocí finálního modelu** a **14) Dávková predikce ze souboru** obsahují interaktivní prvky a mimo prostředí Kaggle nemusí být plně funkční nebo se mohou zobrazovat omezeně.

Z tohoto důvodu je pro úplné ověření funkčnosti notebooku doporučeno použít primárně Kaggle.
